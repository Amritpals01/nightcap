# Nightcap

Nightcap is a sophisticated front-end concept for a premium, after-hours liquor delivery service based in Vancouver. The project showcases a polished, modern user interface designed for a high-end customer experience, complete with dynamic product listings and a fully functional shopping cart.

The website is designed to be operational from 11 PM to 4 AM, catering to late-night needs with a curated selection of spirits, wines, and beers. The UI features a dark, elegant theme, premium typography, and subtle animations to create an immersive feel.

## Features

-   **Elegant & Responsive UI:** A dark, mobile-first design that is fully responsive across all devices. The interface features premium typography and a custom interactive cursor for a polished user experience on desktop.
-   **Interactive Hero Animation:** A captivating hero section powered by `anime.js`, featuring a complex animated SVG dial with parallax effects that respond to mouse movement.
-   **Dynamic Product Showcase:** Product data, including a "Whisky Library," is managed in JavaScript and dynamically rendered onto the page. Prices are calculated with a service markup.
-   **Functional Shopping Cart:** A complete shopping cart system with a slide-out drawer. The cart state is persisted across sessions using `localStorage` and includes features for adding/removing items, adjusting quantities, and clearing the cart.
-   **Live Service Status:** An interactive clock section displays the live Vancouver time and indicates whether the delivery service is currently "Open" or "Closed" based on its operational hours (11 PM - 4 AM).
-   **Accessibility & Performance:** Includes modern web development best practices, such as `prefers-reduced-motion` fallbacks for animations and lazy loading for images.

## Technology Stack

-   **HTML5**
-   **CSS3:** Custom Properties, Flexbox, Grid, Responsive Design, Animations
-   **JavaScript (ES6):** DOM manipulation, event handling, data management
-   **Anime.js:** Used for advanced SVG and element animations in the hero section.
